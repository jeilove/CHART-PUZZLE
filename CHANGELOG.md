# CHANGELOG

## [v1.1.0] - 2026-04-02
### 홈 화면(Home Screen) 대폭 개편 및 프리미엄 디자인 적용
- **홈 화면 레이아웃 재구성**: 
  - **상단 검색 인터페이스**: 기존 중앙 검색창을 제거하고, 상단에 세련된 "Search for news or tickers..." 형태의 고정 검색바와 알림/프로필 아이콘 배치.
  - **시장 지수 카드(Heatmap Placeholders)**: S&P 500, KOSPI, KOSDAQ 등 주요 시장 지수를 한눈에 볼 수 있는 가로 스크롤 카드 레이아웃 구현 (미니 차트 플레이스홀더 포함).
  - **즐겨찾기 그룹 아코디언**: '전체(My List)', '개별 그룹'을 아코디언으로 묶어 효율적으로 종목을 탐색할 수 있는 리스트 UI 구현.
- **디자인 고도화**:
  - **Glassmorphism 적용**: 각 카드와 섹션에 반투명 유리 효과 및 은은한 그라데이션 적용.
  - **애니메이션 강화**: `framer-motion`을 사용하여 아코디언 확장 및 검색 결과 노출 시 부드러운 전환 효과 트리거.
- **실시간 시장 히트맵(Heatmap) 통합**: 
  - **S&P 500 (TradingView)**: TradingView 공식 위젯을 이식하여 상단에 실시간 히트맵 렌더링.
  - **KOSPI/KOSDAQ (D3.js)**: 백엔드 스크래핑 엔진을 통해 시가총액 상위 종목의 실시간 등락 데이터를 수집하고, D3-Hierarchy를 사용하여 커스텀 히트맵 구현.
  - **백엔드 고도화**: `scraper.py`에 네이버 증권 시총 리스트 스크래퍼 추가 및 `main.py`에 전용 API 엔드포인트 개설.
- **주의사항 준수**: 홈 화면 이외의 게임(GAME), 차트(CHART), 데이터 통신 로직 등 기존 핵심 기능은 전혀 건드리지 않고 홈 화면 UI만 정밀 수정함.

## [v1.0.64] - 2026-04-01
### 하단 툴바 완성 및 시스템 안정성 최종 보완
- **하단 툴바(Bottom Tab Bar) 동작 고도화**: 퍼즐 진행 중(정답 확인 후 포함) 퍼즐 탭 아이콘을 클릭하면 게임 상태가 즉시 초기화(`Re-mount`)되어 다시 시작 화면으로 돌아가던 로직 구현.
- **퍼즐 게임 플레이 버그 수정**: 
  - **다시 섞기(Shuffle) 정상화**: 슬롯에 배치된 조각을 포함하여 모든 조각이 강제로 재배치되도록 셔플 로직 전면 수정.
  - **드래그 변형(Deformity) 방지**: 퍼즐 조각을 드래그할 때 가로로 쪼그라드는 그래픽 왜곡 현상을 해결하기 위해 `DragOverlay` 이미지에 `maxWidth: none` 속성 강제 적용.
- **UI/UX 통합 내비게이션**: 차트 상세 화면의 종목명 좌우에 36px 대형 이동 버튼(`<`, `>`) 배치 및 내비게이션 동기화.
- **본문 텍스트 원상복구**: 임의로 번역되었던 `D/W/M`, `Trigger Cloud`, `News Pulse`, `분할..` 등 모든 UI 텍스트를 원래의 영문 및 원본 상태로 완벽 복구.
- **깃허브 동기화**: `master` 브랜치에 "하단 툴바 완성 최종" 메시지와 함께 최종 배포 완료.
