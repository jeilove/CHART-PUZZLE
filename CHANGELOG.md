# CHANGELOG

## [v1.2.1] - 2026-04-02
### 하단 툴바 내비게이션 복구 및 런타임 안정화
- **고질적인 런타임 에러(`createUnhandledError`) 완전 해결**:
  - **TradingView 위젯 최적화**: `page.tsx`에서 TradingView 히트맵 위젯이 렌더링 타이밍 이슈로 `null.querySelector` 에러를 발생시키던 문제를 `useRef`와 `cleanup` 패턴으로 수정하여 해결.
  - **Strict Mode 비활성화**: `next.config.ts`에서 `reactStrictMode: false` 설정을 통해 개발 모드에서의 이중 실행 및 비동기 스크립트 충돌을 원천 차단.
  - **차트 라이프사이클 관리**: `StockChart.tsx`에서 `timeframe` 변경 시 이전 차트 인스턴스에 접근하여 에러를 유발하던 로직을 `useEffect` 내부로 통합하여 안정화.
- **내비게이션 및 UI 동작 보완**:
  - **탭바 반응성 복구**: 타임워프(WARP) 모드에서 하단 탭바의 차트(CHART) 아이콘 클릭 시 반응이 없던 버그를 `isTimeWarpTriggered` 상태 강제 초기화 로직을 통해 해결.
  - **퍼즐 화면 UI 고도화**: 퍼즐 대기 화면을 `UnifiedFlipCard` 구조로 개편하여 트리거 클릭 시 차트 상세 화면과 동일한 3D 플립 애니메이션 적용.
  - **아이콘 위치 최적화**: 사용자의 요청에 따라 퍼즐 화면의 트리거/뉴스 아이콘을 종목 코드 바로 아래로 이동하여 센터 정렬 강화.
  - **불필요한 로딩 표시 제거**: 데이터가 백그라운드에서 로드되는 동안 아이콘 옆에 노출되던 작은 스피너를 제거하여 UI 정숙성 확보.

## [v1.2.0] - 2026-04-02
### 검색 및 즐겨찾기 시스템 전면 고도화
- **검색 UX 혁신**:
  - **다중 선택 체크박스**: 검색 결과 리스트에 체크박스를 도입하여 여러 종목을 한꺼번에 선택할 수 있는 기능 추가.
  - **별표(Star) 트리거**: 개별 종목 클릭 시 차트로 이동하던 기존 로직을 제거하고, '별표' 버튼을 통해 선택된 종목들을 그룹에 일괄 저장하는 직관적인 흐름 구현.
  - **하단 플로팅 UI 제거**: 불필요한 버튼을 삭제하고 별표 버튼 하나로 모든 저장 액션을 통합하여 화면을 깔끔하게 정리.
- **프리미엄 그룹 관리**:
  - **'Select lists' 바텀 시트**: `framer-motion` 기반의 세련된 바텀 시트 UI를 통해 기존 즐겨찾기 그룹 선택 및 신규 그룹 생성을 한 곳에서 처리.
  - **일괄 추가 로직**: 선택된 다수의 종목을 특정 그룹에 한 번에 저장하는 백엔드 연동 로직 안정화.
- **데이터 정밀도 향상 (Industry Data Audit)**:
  - **핵심 테마 매핑**: **조선, 반도체, 2차전지, 자동차** 등 주요 투자 테마에 대해 업종 데이터를 전수 조사하고 수동 매핑을 통해 검색 정확도 대폭 향상.
  - **지주사 자동 분류**: '홀딩스', '지주' 키워드가 포함된 종목들을 '지주회사' 카테고리로 자동 통합.
- **시스템 안정성 및 운영 정책**:
  - **프로세스 자동 클린업**: `run_puzzle.bat` 실행 시 기존에 좀비화된 Node, Python 프로세스를 자동으로 강제 종료(`taskkill`)하도록 보완.
  - **글로벌 지침 업데이트**: 모든 프로젝트에서 배치 파일 생성 시 자동 클린업 로직을 포함하도록 `GEMINI.md` 전역 원칙 추가.
  - **빌드 오류 해결**: `AnimatePresence` 및 `HomeIcon` 컴포넌트 구조 결함을 수정하여 Next.js 빌드 안정성 확보.


## [v1.1.0] - 2026-04-02
### 홈 화면(Home Screen) 대폭 개편 및 프리미엄 디자인 적용
- **홈 화면 레이아웃 재구성**: 
  - **상단 검색 인터페이스**: 기존 중앙 검색창을 제거하고, 상단에 세련된 "Search for news or tickers..." 형태의 고정 검색바와 알림/프로필 아이콘 배치.
  - **시장 지수 카드(Heatmap Placeholders)**: S&P 500, KOSPI, KOSDAQ 등 주요 시장 지수를 한눈에 볼 수 있는 가로 스크롤 카드 레이아웃 구현 (미니 차트 플레이스홀더 포함).
  - **즐겨찾기 그룹 아코디언**: '전체(My List)', '개별 그룹'을 아코디언으로 묶어 효율적으로 종목을 탐색할 수 있는 리스트 UI 구현.
- **디자인 고도화**:
  - **Glassmorphism 적용**: 각 카드와 섹션에 반투명 유리 효과 및 은은한 그라데이션 적용.
  - **애니메이션 강화**: `framer-motion`을 사용하여 아코디언 확장 및 검색 결과 노출 시 부드러운 전환 효과 트리거.
- **실시간 시장 히트맵(Heatmap) 통합**: 
  - **S&P 500 (TradingView)**: TradingView 공식 위젯을 이식하여 상단에 실시간 히트맵 렌더링.
  - **KOSPI/KOSDAQ (D3.js)**: 백엔드 스크래핑 엔진을 통해 시가총액 상위 종목의 실시간 등락 데이터를 수집하고, D3-Hierarchy를 사용하여 커스텀 히트맵 구현.
  - **백엔드 고도화**: `scraper.py`에 네이버 증권 시총 리스트 스크래퍼 추가 및 `main.py`에 전용 API 엔드포인트 개설.
- **서버 포트 및 실행 환경 최적화**: 
  - **포트 고정**: `frontend/package.json`의 `dev` 스크립트에 `--port 3000`을 추가하여 포트 충돌 방지 및 고정 포트 사용.
  - **배치 파일 동기화**: `run_puzzle.bat`, `run_project.bat`, `run_dev.bat` 파일의 브라우저 실행 URL을 모두 `http://localhost:3000`으로 통일.
  - **안정성 강화**: 좀비 프로세스로 인한 포트 점유 이슈 해결을 위해 프로세스 클린업 가이드 준수.
- **주의사항 준수**: 홈 화면 이외의 게임(GAME), 차트(CHART), 데이터 통신 로직 등 기존 핵심 기능은 전혀 건드리지 않고 홈 화면 UI만 정밀 수정함.

## [v1.0.64] - 2026-04-01
### 하단 툴바 완성 및 시스템 안정성 최종 보완
- **하단 툴바(Bottom Tab Bar) 동작 고도화**: 퍼즐 진행 중(정답 확인 후 포함) 퍼즐 탭 아이콘을 클릭하면 게임 상태가 즉시 초기화(`Re-mount`)되어 다시 시작 화면으로 돌아가던 로직 구현.
- **퍼즐 게임 플레이 버그 수정**: 
  - **다시 섞기(Shuffle) 정상화**: 슬롯에 배치된 조각을 포함하여 모든 조각이 강제로 재배치되도록 셔플 로직 전면 수정.
  - **드래그 변형(Deformity) 방지**: 퍼즐 조각을 드래그할 때 가로로 쪼그라드는 그래픽 왜곡 현상을 해결하기 위해 `DragOverlay` 이미지에 `maxWidth: none` 속성 강제 적용.
- **UI/UX 통합 내비게이션**: 차트 상세 화면의 종목명 좌우에 36px 대형 이동 버튼(`<`, `>`) 배치 및 내비게이션 동기화.
- **본문 텍스트 원상복구**: 임의로 번역되었던 `D/W/M`, `Trigger Cloud`, `News Pulse`, `분할..` 등 모든 UI 텍스트를 원래의 영문 및 원본 상태로 완벽 복구.
- **깃허브 동기화**: `master` 브랜치에 "하단 툴바 완성 최종" 메시지와 함께 최종 배포 완료.
