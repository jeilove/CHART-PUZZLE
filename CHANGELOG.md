## [v1.6.8] - 2026-04-04

## [v1.6.4] - 2026-04-04
### 스마트 즐겨찾기 토글(Smart Toggle) 및 복구 로직 도입

  - 새로운 종목 추가 시에만 그룹 지정 드로어를 열어 불필요한 조작을 최소화.
- **데이터 정합성**:
  - `lastRemovedFavoriteLocation` 상태를 신설하여 종목별 직전 저장 상세 데이터를 추적.

## [v1.6.2] - 2026-04-03
### 검색 결과 그룹화 및 아코디언 인터페이스 도입
- **검색 결과 구조화**:
  - 검색된 종목 중 즐겨찾기 그룹에 속한 종목은 해당 **그룹명 아코디언** 내부에 자동으로 배치되도록 개선.

- **아코디언 애니메이션**:
  - `framer-motion`과 `AnimatePresence`를 활용하여 검색 결과 내 그룹 열기/닫기 시 부드러운 전환 효과 적용.
- **코드 구조 개선**:
  - 검색 결과 항목을 `SearchResultItem` 컴포넌트로 분리하여 UI 일관성 유지 및 유지보수성 향상.

## [v1.6.0] - 2026-04-03
### 홈 검색 풀화면 전환 및 지수 감쇠 워드클라우드 통합
- **홈 화면 검색 혁신**:
  - **풀화면 검색 모드**: 기존 오버레이 검색에서 독립적인 풀화면(`isSearchFullScreen`) 검색 환경으로 전환.
  - **내비게이션**: 상단 왼쪽에 '이전으로(`<-`) 메뉴를 추가하여 홈 화면으로의 복귀 직관성 확보.
- **검색 결과 레이아웃 고도화 (Watchlist 스타일)**:
  - **프리미엄 리스트**: 종목명을 강조하고 종목 코드를 생략하여 가독성 높인 "PORTFOLIOS & WATCHLISTS" 스타일 적용.
  - **4대 액션 아이콘 통합**: 종목별 차트, 퍼즐, 타임워프, 워드클라우드 아이콘 전진 배치.
  - **실시간 데이터 시뮬레이션**: 검색 결과에서 주가 및 등락률(%) 정보를 즉시 확인할 수 있도록 백엔드 연동 강화.
  - **즐겨찾기 카드 통합**: 우측 별표 아이콘을 통해 검색 화면에서 즉시 즐겨찾기 상태 변경 가능.
- **워드클라우드(Trigger Cloud) 기능 통합**:
  - **지수적 감쇠 모델 적용**: 시계열 가중치($e^{-\lambda \cdot \Delta t}$)가 적용된 트리거 리포트 분석 데이터를 워드클라우드 아이콘과 연동.
  - **자동 플립 로직**: 검색 결과에서 워드클라우드 아이콘 클릭 시, 차트 화면으로 이동함과 동시에 즉시 리버스(Back) 면이 노출되도록 `initialFlipped` 로직 구현.
- **기타 개선 사항**:
  - **버전 업데이트**: UI 및 콘솔 로그 `v1.6.0` 반영.
  - **UI 최적화**: 검색 화면 진입/이탈 시 `framer-motion` 기반의 스케일 및 페이드 애니메이션 적용.

## [v1.5.4] - 2026-04-03
### 전수 조사 엔진 고도화 및 350종목 감성 분석 재가동
- **주요 업데이트**: 350종목 전수 분석 엔진 (KOSPI 200 + KOSDAQ 150)
- **감성 로직**: 시계열 감쇠 가중치(Exponential Decay) 도입 ($e^{-\lambda \cdot \Delta t}$, 반감기 14일)
- **데이터 무결성**: Jina Reader 800라인 제한 해제 및 리포트 전수 텍스트 스캔
- **안정성**: 구글 뉴스 RSS 전환 및 BeautifulSoup 타임아웃 강화로 무한 루프 원천 차단
- **UI/API 연동**:
    - `trigger_report.json` 기반의 350종목 분석 결과 API 연동 (`fetch_trigger_summary`)
    - 트리거 클라우드 및 감성 타임라인 시각화 최적화를 위한 점수 정규화(Normalization) 적용
    - 감성 진화 타임라인(Timeline) 바 높이 스케일링 상향 및 불필요한 라벨 제거
    - 전수 조사 기반의 데이터 다양성 확보 (긍정/부정/변화 Top 20 상시 유지)
- **분석 자동화 및 성능 최적화**:
  - **350종목 일괄 분석**: KOSPI 200 및 KOSDAQ 150 전수 조사 대상 확정 및 병렬 처리 엔진(ThreadPoolExecutor) 최적화.
  - **무한 루프 방지**: 네이버 뉴스/리포트 크롤링 시 발생하던 프로세스 고착 현상을 해결하기 위해 구글 뉴스 RSS 전환 및 BeautifulSoup 파싱 타임아웃 강화.
- **데이터 시각화 데이터셋 생성**:
  - **상위 20 종목 선별**: 350개 종목 중 가장 완결성 있는 감성 시그널을 보유한 상위 20개 종목(Positive, Negative, Change, Trend) 추출 및 `trigger_report.json` 업데이트.

## [v1.3.0] - 2026-04-02
### 트리거 분석 대시보드 (Trigger Pulse) 도입 및 통합 내비게이션 확장
- **신규 '트리거(TRIGGER)' 분석 화면 추가**:
  - **3종 전용 스톡 클라우드**:

    - **부정 트리거 클라우드**: 시장 내 부정적인 키워드가 집중된 종목들을 리스크 관리 차원에서 노출.
    - **변화량(Change) 클라우드**: '급증', '감소', '쇼티지', '증설' 등 수량적 변화를 나타내는 핵심 키워드가 포착된 종목들을 별도 분류.
  - **감성 지수 타임라인(Timeline View)**: 주요 종목들의 감성 지수(Sentiment Score) 변화를 시계열 그래프로 구현하여 시장 심리 추세 파악 지원.
- **하단 탭바 기능 확장**:
  - **트리거 아이콘 추가**: 탭바 우측 끝에 'v3_trigger.png' 아이콘을 배치하여 분석 화면으로 즉시 이동 가능한 5번째 탭 신설.
  - **통합 내비게이션(Global View State)**: `page.tsx`의 뷰 상태 시스템(`HOME`, `GAME`, `CHART`, `TRIGGER`)을 확장하여 끊김 없는 화면 전환 구현.
- **백엔드 분석 엔진 고도화**:
  - **일괄 분석 API (`/api/trigger/summary`)**: 시가총액 상위 종목들을 한 번에 스캔하고 트리거를 추출하여 대시보드 데이터를 생성하는 전용 엔드포인트 구축.
  - **'변화' 키워드 사전 강화**: `scraper.py`의 `TRIGGER_KEYWORDS`에 '수주 확대', '재고 감소', '가동률 상승' 등 변화 포착용 키워드 대거 추가.
- **운영 환경 및 배치 파일 최적화**:
  - **프로세스 클린업 강화**: `run_dev.bat` 실행 시 기존에 동작 중인 Node, Python, Uvicorn 프로세스를 자동으로 감제 종료하고 포트를 점유한 좀비 프로세스를 제거하는 로직 적용.
  - **버전 정보 업데이트**: UI 상단 및 콘솔에 `v0.6.1` -> `v1.3.0` 버전 정보 반영.

- **내비게이션 및 UI 동작 보완**:
  - **탭바 반응성 복구**: 타임워프(WARP) 모드에서 하단 탭바의 차트(CHART) 아이콘 클릭 시 반응이 없던 버그를 `isTimeWarpTriggered` 상태 강제 초기화 로직을 통해 해결.
