# 🧩 마스터 작업 계획서 : 주식 차트 퍼즐 프로젝트 (v1.0)
**상태**: 사용자 피드백 반영 및 단계별 고도화 중

이 문서는 초기 제품 요구사항 정의서(PRD)부터 현재까지 사용자님께서 주신 모든 피드백과 기능 구현 현황을 통합하여 기록합니다. 프로젝트의 '기준점'이 되는 문서로, 중복되거나 누락된 계획을 방지합니다.

---

## 1. 프로젝트 비전 및 핵심 기능 (Initial Master Plan)
- **핵심 가치**: 차트 조각을 맞추며 주가 패턴을 학습하고, 예측 퀴즈(타임 워프)를 통해 실전 감각을 고도화함.
- **주요 기능**:
    1. **블라인드 모드**: 종목명을 가린 채 차트만 보고 퍼즐 진행.
    2. **정교한 직소 엔진**: 탭/슬롯 형태의 퍼즐 및 자석 흡착(Snapping) 효과.
    3. **타임 워프 퀴즈**: 퍼즐 완성 후 5일간의 주가 방향(상등/하락) 예측. (중요 피드백 반영 예정)
    4. **실시간 뉴스 워드 클라우드**: 차트 배경에 뉴스 기반 가중치 분석 결과 시각화.
    5. **실데이터 적용**: 최종적으로 Mock 데이터를 실제 주식 API 데이터로 전환. (중요 피드백 반영 예정)
