# 📄 제품 요구사항 정의서 (PRD) : 주식 차트 조각 그림 맞추기 게임

## 1. 개요 (Product Overview)
본 앱은 KOSPI 200 및 KOSDAQ 150 종목의 실제 일봉 차트를 퍼즐 형태로 맞추며 주가 흐름 패턴을 학습하는 모바일 에듀테인먼트 게임입니다. 배경에 실시간 뉴스 기반 워드 클라우드를 시각적으로 제공하여 종목의 현재 분위기를 파악하고, 퍼즐 완성 후 향후 5일의 주가 방향을 예측하는 퀴즈를 통해 실전 감각을 기릅니다.

## 2. 타겟 유저 (Target Audience)
- 주식 투자에 관심이 많으나 차트 보는 법이 낯선 초보 투자자
- 주가 패턴을 게임처럼 재밌게 학습하고 싶은 2030 개인 투자자
- 주식 시장의 트렌드와 뉴스를 시각적으로 빠르게 파악하고 싶은 유저

## 3. 핵심 기능 명세 (Key Features)

### 3.1. 게임 모드
1. **블라인드 모드 (Blind Challenge)**
   - **설명:** 종목명을 가린 채 퍼즐 게임 진행.
   - **종목 선정:** 전일 상승률 상위 30개 종목 중 랜덤 1개 선택.
   - **조건:** 퍼즐 완료 전까지 티커/종목명 비공개. 완료 시 공개.
2. **검색 모드 (Direct Selection)**
   - **설명:** 유저가 직접 관심 있는 종목을 검색하여 게임 진행.

### 3.2. 차트 및 퍼즐 로직
- **차트 데이터:** 최근 65일 영업일 데이터 활용 (최초 퍼즐에는 최근 5일을 제외한 60일치 데이터만 표시).
- **지표 포함:** 캔들스틱, 이동평균선(5, 10, 20, 60일), 하단 거래량.
- **퍼즐 분해:** N x M 그리드 (예: 4x4, 16조각)로 차트 이미지를 분할하여 랜덤 셔플.
- **인터랙션:** Drag & Drop 방식으로 조각 이동. 이동 횟수 및 소요 시간 기록.

### 3.3. 타임 워프 예측 (Time Warp)
- **설명:** 퍼즐 완성 직후, 제외되었던 최근 5일간의 주가 방향(상승/하락)을 맞추는 퀴즈 제공.
- **결과:** 유저 선택 후 실제 최근 5일 치 차트가 애니메이션과 함께 덧붙여지며 정답 여부 공개.

### 3.4. 워드 클라우드 배경 시스템 (Real-time Word Cloud)
- **설명:** 종목명 확정 시, 즉각적으로 최근 뉴스(24시간~1주일)를 분석하여 워드 클라우드 생성.
- **감성 분석(컬러 코딩):** 긍정 뉴스 지배적일 시 붉은색/금색 톤, 부정 뉴스 지배적일 시 파란색/회색 톤 적용.
- **UI 적용:** 퍼즐 맞추기 화면의 배경 캔버스로 사용 (투명도 조절로 차트 시인성 확보).

## 4. 기술 스택 (Technical Specifications)
| 구분 | 기술 스택 | 설명 |
| :--- | :--- | :--- |
| **Frontend** | React Native (또는 Flutter) | 모바일 크로스 플랫폼 앱 개발 |
| **Chart UI** | Lightweight Charts | 캔버스 기반 렌더링, `toDataURL()`을 통한 이미지 추출 및 조각화에 최적 |
| **Backend** | Python (FastAPI/Flask), Render | 워드 클라우드 파이프라인 구축 및 API 제공 |
| **NLP/Data** | KoNLPy, NLTK, `wordcloud` | 텍스트 전처리(불용어 제거) 및 이미지 생성 |
| **Open API** | Naver News API, Naver/Kakao Finance | 주가 데이터 크롤링 및 뉴스 헤드라인 수집 |

## 5. 제약 사항 및 고려 사항
- 모바일 환경에서의 Drag & Drop 퍼포먼스 최적화 필요 (Canvas 위에서의 이벤트 핸들링).
- 워드 클라우드 생성 시간 지연(1~3초) 시 UX 처리를 위한 로딩 스켈레톤 또는 애니메이션 필수.
- 뉴스 API의 광고성, 중복 키워드 필터링을 위한 강력한 불용어(Stopwords) 사전 구축 필요.
