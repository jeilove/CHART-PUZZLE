# 📱 화면 설계서 (Storyboard) : 주식 차트 조각 그림 맞추기

## Screen 1: 스플래시 & 메인 화면 (Home)
- **UI 요소:**
  - 타이틀 로고 ("차트 조각 모음" 가칭)
  - [블라인드 모드 시작] 버튼 (전일 상승률 상위 종목 랜덤 매칭)
  - [종목 검색하여 시작] 검색 바 (자동 완성 지원 - 코스피200/코스닥150 제한)
  - 내 최고 기록 / 누적 예측 성공률 요약 패널

## Screen 2: 로딩 화면 (Data Fetching & Processing)
- **UI 요소:**
  - "차트 데이터를 불러오고 최신 뉴스를 분석 중입니다..." 텍스트
  - 맥박이 뛰거나 캔들이 오르락내리락하는 로딩 애니메이션
  - *백그라운드 로직:* 주가 데이터 로드 -> 차트 이미지 렌더링 -> 뉴스 크롤링 -> 워드 클라우드 생성

## Screen 3: 퍼즐 게임 화면 (Puzzle Play)
- **UI 요소:**
  - **상단:** 타이머(00:00), 이동 횟수(0회) 표기. 블라인드 모드일 경우 종목명은 "???", 검색 모드일 경우 "삼성전자".
  - **배경:** 서버에서 받아온 워드 클라우드 이미지 (투명도 30~40% 적용, 긍정/부정 톤 반영).
  - **메인 영역:** 4x4 그리드로 쪼개져 무작위로 섞인 차트 조각들.
  - **하단:** [원본 차트 살짝 보기] 힌트 버튼 (누르고 있는 동안만 투명하게 원본 표시).
- **인터랙션:** 터치 후 드래그하여 조각의 위치를 교환(Swap).

## Screen 4: 퍼즐 완료 & 타임 워프 퀴즈 (Time Warp Prediction)
- **UI 요소:**
  - 펑 터지는 폭죽 애니메이션과 함께 퍼즐 완성.
  - (블라인드 모드인 경우) 화면 중앙에 "이 종목은 [SK하이닉스] 였습니다!" 팝업 노출.
  - 차트 우측 끝(빈 공간)을 하이라이트하며 팝업 등장:
    - **"과연 이후 5일 동안 주가는 어떻게 되었을까요?"**
  - **버튼:** [📈 상승했다] / [📉 하락했다]

## Screen 5: 최종 결과 화면 (Result)
- **UI 요소:**
  - 사용자의 선택 직후, 가려졌던 5일 치 캔들 데이터가 오른쪽에서 슬라이드 인(Slide-in) 되며 애니메이션으로 차트 완성.
  - **정답 여부 표기:** "정답입니다!" or "아쉽네요!"
  - **게임 기록 요약:**
    - 퍼즐 완성 시간 / 이동 횟수
    - 해당 종목의 지난 5일간 실제 수익률 (%)
  - **하단 액션 버튼:** [다른 종목으로 다시 하기] / [내 SNS에 결과 공유하기]
