# 🧩 UX 개편: 직소 퍼즐 드래그 앤 드랍 (Phase 3)

사용자가 요청한 대로 퍼즐 조각을 외부 풀(Pool)에 배치하고, 드래그 앤 드랍으로 그리드에 맞추는 방식으로 개편합니다.

## 📋 작업 목록
- [ ] **데이터 구조 변경**: 조각의 위치(x, y), 고정 여부(isPlaced)를 포함하는 객체 배열로 상태 관리 변경
- [ ] **초기 배치 로직 구현**: 퍼즐 보드 외곽 영역에 랜덤하게 조각 분산 배치 (겹침 허용)
- [ ] **DndKit 리팩토링**:
    - [ ] `SortableContext` 제거 및 `useDraggable` / `useDroppable` 기반으로 변경
    - [ ] 드랍 시 정답 위치 근처일 경우 자동 흡착(Snapping) 로직 구현
- [ ] **UI/UX 개선**:
    - [ ] 퍼즐 보드에 가이드용 배경(Grayscale Ghost Image) 추가
    - [ ] 조각 드래그 시 최상단 출력(Z-Index) 및 부드러운 애니메이션
    - [ ] 정답 안착 시 시각적 피드백 강화
- [ ] **완료 검증**: 모든 조각이 제자리에 배치되었을 때 게임 종료 처리
- [ ] **기록 업데이트**: `CHANGELOG.md` 및 `debug_history.md` 기록

## 🎨 디자인 가이드
- **색상**: 은은한 다크 모드 (`#0d1117`, `#4B4646`)
- **효과**: 유리창 효과(Glassmorphism), 매끄러운 그라데이션
- **애니메이션**: `framer-motion`을 이용한 부드러운 등장 및 안착 효과
