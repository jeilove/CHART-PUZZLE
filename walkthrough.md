# Walkthrough: Stock Chart Puzzle - News Pulse Integration (v1.0.40)

본 문서에서는 Stock Chart Puzzle의 뉴스 펄스(News Pulse) 기능 통합 및 아이콘 기반 UI 정밀 보정 과정을 상세히 안내합니다.

## 1. 뉴스 펄스(News Pulse) 통합 및 아이콘화
사용자의 요청에 따라 파편화되어 있던 뉴스 확인 버튼들을 하나로 통합하고 고해상도 아이콘으로 대체했습니다:
- **헤더 통합 배치**: 차트 헤더의 종목코드 우측, '트리거' 아이콘 바로 옆에 '뉴스 펄스' 아이콘(`v18_pulse.png`)을 배치했습니다.
- **원형 마스킹(Rounded Masking)**: 사각형 아이콘 이미지를 CSS `rounded-full`과 `object-cover` 속성을 사용하여 세련된 원형으로 렌더링했습니다.
- **간격 최적화**: 트리거와 뉴스 아이콘 사이의 간격을 기존 `gap-3`(12px)에서 `gap-1.5`(6px)로 50% 축소하여 시각적 일체감을 높였습니다.

## 2. 전역 뉴스 오버레이(Global News Overlay) 시스템
기능적 제약 없이 어느 화면에서든 뉴스를 확인할 수 있는 지능형 오버레이 시스템을 구축했습니다:
- **전 상태 접근 가능**: '블라인드 챌린지 시작', '퍼즐 플레이 중', '퀴즈 결과 대기' 등 모든 게임 상태에서 헤더의 뉴스 아이콘을 클릭하면 즉시 뉴스 창이 나타납니다.
- **최상위 레이어(`z-8000`)**: 뉴스 창의 Z-Index를 상향하고 `fixed` 좌표계를 적용하여, 퀴즈 모달과 같은 레이어 위에서도 가려짐 없이 노출됩니다.
- **지능형 자동 닫기**: 트리거 아이콘을 클릭하여 카드 뒷면(분석 결과)으로 전환될 때, 열려 있던 뉴스 창을 자동으로 닫아 정보 중첩을 방지합니다.

## 3. 정답 확인 버튼 인터랙션 고도화
퍼즐 완료 및 확인 과정에서의 타격감과 시각적 피드백을 강화했습니다:
- **Hover & Scale**: '정답 확인' 버튼에 마우스 오버 시 `scale-105` 확대 효과와 함께 그림자가 더욱 깊어지며 배경색이 변하는 역동적인 피드백을 추가했습니다.
- **Transition**: `duration-300`의 부드러운 전환을 통해 프리미엄 UI에 걸맞은 조작감을 제공합니다.

## 4. 레이아웃 정밀 튜닝
- **퀴즈 문구 간격 보정**: "타임 워프 퀴즈 도전" 문구와 거래량 차트 사이의 불필요한 공백을 50% 제거하여 레이아웃 밀도를 높였습니다.
- **후면(Back) 헤더 정제**: 분석 결과 화면에서 종목명 옆에 발생하던 중복 아이콘 세트를 제거하여 데이터 가독성을 확보했습니다.
