# 🗺️ 프로젝트 단계별 구축 가이드 (Walkthrough)

이 문서는 '주식 차트 조각 그림 맞추기' 프로젝트의 각 단계별 마일스톤과 구현 현황을 추적합니다.

## 🏁 Phase 1: 기본 인프라 및 차트 엔진 (완료)
- [x] Next.js(App Router) 기반 프론트엔드 환경 구축
- [x] `Lightweight Charts`를 이용한 실시간 주식 차트 렌더링
- [x] 캔버스 데이터 추출(`toDataURL`) 및 이미지 조각화 기본 로직

## 🛠️ Phase 2: 퍼즐 엔진 고도화 - 직소 스타일 (완료)
- [x] **Bezier Curve 기반 직소 렌더링**: 단순 사각형이 아닌 실제 퍼즐 모양(Tab/Slot) 구현
- [x] **픽셀 매칭 최적화**: 조각 간 연결 부위의 1px 틈새 제거 (`maxWidth: none` 적용)
- [x] **고퀄리티 렌더링**: `#4B4646` 테마 색상 및 안티-앨리어싱 처리

## 🕹️ Phase 3: 드래그 앤 드랍 UX 개편 (완료)
- [x] **자유 배치 시스템**: 보드 바깥에서 조각을 끌어오는 '자유 드롭' 방식 전환
- [x] **스내핑(Snapping)**: 정답 위치 근처 드롭 시 부드러운 자석 효과로 안착
- [x] **가이드 시스템**: 보드 배경에 흑백 투명 차트 이미지(Ghost Image) 배치

## ✨ Phase 4: 프리미엄 폴리싱 및 사용자 편의 (완성 단계)
- [x] **난이도 선택**: 3x3(9개), 4x4(16개), 5x5(25개) 개수 선택 UI 구현
- [x] **디자인 최적화**: 버튼 및 선택 박스 크기 슬림화 (70%, 60% 축소)
- [x] **뉴스 펄스(News Pulse)**: 실시간 감성 분석 지표 및 태그 클라우드 (레이어 최상단 배치)
- [x] **버그 수정**: 정답 확인 후 '다시 섞기'를 통한 게임 재시작 로직 완비

## 🚀 Phase 5: 타임 워프 퀴즈 및 게임 로직 고도화 (예정)
- [ ] **타임 워프(Time Warp)**: 퍼즐 완성 직후 최근 5일간의 주가 방향(상승/하락) 예측 퀴즈 구현
- [ ] **애니메이션 연출**: 예측 후 가려졌던 5일치 캔들이 오른쪽에서 슬라이드 인 되며 정답 공개
- [ ] **종목 정보 공개**: 블라인드 모드일 경우 종목명 및 최종 결과 요약 팝업 노출

## 🔌 Phase 6: 실데이터 연동 및 시스템 안정화 (예정)
- [ ] **실제 데이터 적용**: 목업(Mock) 데이터를 실제 KOSPI/KOSDAQ API 데이터로 전면 교체
- [ ] **워드 클라우드 동기화**: 뉴스 API 기반 실시간 키워드 분석 및 배경 생성 연동
- [ ] **모바일 최적화**: 실제 디바이스 환경에서의 터치 감도 및 레이아웃 최종 튜닝
- [ ] **배포 가이드**: Vercel/Supabase 등 실제 운영 환경 배포 및 데이터 동기화 점검
